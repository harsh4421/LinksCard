# Social Media Hover Cards

Interactive social media cards with:
- Hover glow effect
- Live GitHub follower count
- Instagram & LinkedIn links

## Features
- HTML, CSS, JavaScript
- Font Awesome icons
- GitHub API integration

## Setup
1. Replace usernames in `index.html` and `script.js`
2. Open `index.html` in browser

## Note
Instagram & LinkedIn follower counts require backend APIs.
